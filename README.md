# 🎓 Smart Attendance Risk Predictor

## 📌 Project Overview

Smart Attendance Risk Predictor is a Python-based Data Analytics project that analyzes student attendance records and identifies attendance risks.
The project provides meaningful insights about student attendance performance and helps identify students who need improvement.

## 🚀 Features

✅ Overall Attendance Percentage Analysis
✅ Practical Attendance Analysis
✅ Attendance Risk Prediction
✅ Exam Eligibility Checking
✅ Classes Required to Reach 75% Attendance
✅ Safe Bunks Calculation
✅ Top Performing Students Identification
✅ At-Risk Student Detection
✅ Department-wise Attendance Analysis
✅ Data Visualization
✅ Automated Attendance Report Generation
